## Wires
Starter file for wireframe design systems.

## Artboards / Frames

Includes four artboards:
* 360px
* 768px
* 1280px
* 1440px

## Layout

Follows an 8px grid with a 4px baseline grid.

* 360px
  * columns: 4 / 60px
  * margins: 24px
  * gutters: 24px
* 768pt
  * columns: 8 / 76px
  * margins: 24px
  * gutters: 16px
* 1280pt
  * columns: 12 / 88px
  * margins: 24px
  * gutters: 16px
* 1440pt
  * columns: 12 / 80px
  * margins: 64px
  * gutters: 32px

## Typography

Base font size set to 16px, line height of 1.5 and scale factor of 1.618.

  line-height: 24px;
}
